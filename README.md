# 🌟 KidQuiz Pro: Full-Stack Educational Platform

KidQuiz Pro is a responsive, high-performance web application designed to gamify learning for children. Built with **Python (Flask)** and **JavaScript**, the platform features a Single Page Application (SPA) architecture, real-time feedback, and persistent progress tracking.

## ✨ Key Features
- **Dynamic Question Engine:** Randomly selects 15 questions from a JSON-based database for unique sessions.
- **Real-Time Interactive Timer:** 15-second response window per question implemented with client-side JavaScript.
- **Instant Visual Feedback:** Glow-effects (Green/Red) providing immediate reinforcement for learning.
- **Client-Side Persistence:** High-score tracking using the Web Storage API (localStorage).
- **Responsive Design:** Optimized for both mobile and desktop browsers using CSS3 Flexbox.

## 🛠️ Tech Stack
- **Backend:** Python 3.x, Flask
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Data Management:** JSON Data Serialization
